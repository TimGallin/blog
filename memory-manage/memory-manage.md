###C/C++内存分布

以32位4G内存为例，内存地址由高位至地位
>**Kernel Space**
内核空间，这部分内存有操作系统占用

>**Stack**
栈空间，逻辑连续的FIFO先入先出栈内存，主要用于存储程序运行过程中的临时变量，函数出入参以及返回结果相关的临时变量

>**MemoryMapping**
内存映射空间，进程内有多线程共享内存存在时，这部分内存位于堆和栈之间

>**Heap**
堆空间，逻辑不连续的内存空间，通常用于存储Malloc分配的内存空间

>**BSS(Block Started By Symbol)**
存储未初始化的全局变量与静态变量

>**Data Segment**
存储已经初始化的全局变量和静态变量

>**Code Segment**
也称为TEXT段，存储程序资源，包括编码，资源等
